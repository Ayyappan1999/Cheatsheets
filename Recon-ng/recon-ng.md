# RECON-NG

Certainly! Here's a cheatsheet for the `recon-ng` tool, a powerful framework for reconnaissance and information gathering:

```
+-------------------------------------------------------+
|                     recon-ng Tool                     |
+-------------------------+-----------------------------+
|        Command          |         Description         |
+-------------------------+-----------------------------+
| recon-ng                | Launch the recon-ng         |
|                         | framework                   |
+-------------------------+-----------------------------+
| marketplace search      | Search for available        |
| <module_name>           | modules in the marketplace  |
+-------------------------+-----------------------------+
| marketplace info        | Display information about   |
| <module_name>           | a specific module           |
+-------------------------+-----------------------------+
| marketplace install     | Install a module from the   |
| <module_name>           | marketplace                 |
+-------------------------+-----------------------------+
| modules search          | Search for available        |
| <module_name>           | modules in the recon-ng    |
|                         | framework                   |
+-------------------------+-----------------------------+
| modules info            | Display information about   |
| <module_name>           | a specific module           |
+-------------------------+-----------------------------+
| modules load            | Load a specific module      |
| <module_name>           | into the framework          |
+-------------------------+-----------------------------+
| options set             | Set an option value for a   |
| <option_name> <value>   | loaded module               |
+-------------------------+-----------------------------+
| options show            | Show the current values of  |
|                         | options for a loaded module |
+-------------------------+-----------------------------+
| keys add                | Add an API key for a        |
| <service_name> <key>    | specific service            |
+-------------------------+-----------------------------+
| keys list               | List all configured API     |
|                         | keys for services           |
+-------------------------+-----------------------------+
| keys remove             | Remove a configured API     |
| <service_name>          | key for a specific service  |
+-------------------------+-----------------------------+
| show workspaces         | Show available workspaces   |
|                         | in the framework            |
+-------------------------+-----------------------------+
| workspaces select       | Select a specific workspace |
| <workspace_name>        | to work with                |
+-------------------------+-----------------------------+
| workspaces add          | Add a new workspace to the  |
| <workspace_name>        | framework                   |
+-------------------------+-----------------------------+
| workspaces delete       | Delete a specific workspace |
| <workspace_name>        | from the framework          |
+-------------------------+-----------------------------+
| run                      | Run the loaded module and   |
|                         | execute its functionality   |
+-------------------------+-----------------------------+
| back                    | Return to the main menu     |
+-------------------------+-----------------------------+
| help                    | Display help menu           |
+-------------------------+-----------------------------+
```

These are the commonly used commands in the `recon-ng` framework. Use them to load modules, configure options, manage workspaces, and perform reconnaissance tasks using various modules available in the framework.
